HTML - Hyper Text Markup Language

HTML is the standard markup language used for creating websites
Each HTML element is defined by using "< >" tags

CSS - Cascading Style Sheet

CSS is used for editing where and how HTML elements are displayed on the website.
There are 3 types of CSS: External, Internal and Inline.


External CSS comes from a separate file linked to the HTML document.

Internal CSS is written in the same document as the HTML, but located within the style tags.

Inline CSS is small bits of CSS witten at the start of the HTML.

JS - Javascript

Javascript is another programming languages used for web design.
It can be used to edit CSS or HTML and manipulate data.
